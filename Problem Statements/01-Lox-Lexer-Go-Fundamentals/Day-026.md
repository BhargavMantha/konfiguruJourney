# Lexer Implementation Patterns

## Core Scanner Structure

```go
type Scanner struct {
    source  string
    tokens  []Token
    start   int
    current int
    line    int
    errors  []string
}
```

## Essential Methods

**Advancing:**
```go
func (s *Scanner) advance() byte {
    char := s.source[s.current]
    s.current++
    return char
}
```

**Lookahead:**
```go
func (s *Scanner) peek() byte {
    if s.isAtEnd() {
        return 0
    }
    return s.source[s.current]
}

func (s *Scanner) match(expected byte) bool {
    if s.isAtEnd() || s.source[s.current] != expected {
        return false
    }
    s.current++
    return true
}
```

## Common Patterns

**Two-character tokens:**
```go
case '!':
    if s.match('=') {
        s.addToken(BANG_EQUAL)
    } else {
        s.addToken(BANG)
    }
```

**String literals:**
```go
func (s *Scanner) scanString() {
    for s.peek() != '"' && !s.isAtEnd() {
        if s.peek() == '\n' {
            s.line++
        }
        s.advance()
    }

    if s.isAtEnd() {
        s.addError("Unterminated string.")
        return
    }

    s.advance() // Closing "
    value := s.source[s.start+1 : s.current-1]
    s.addTokenLiteral(STRING, value)
}
```

**Numbers:**
```go
func (s *Scanner) scanNumber() {
    for isDigit(s.peek()) {
        s.advance()
    }

    if s.peek() == '.' && isDigit(s.peekNext()) {
        s.advance() // Consume '.'
        for isDigit(s.peek()) {
            s.advance()
        }
    }

    text := s.source[s.start:s.current]
    value, _ := strconv.ParseFloat(text, 64)
    s.addTokenLiteral(NUMBER, value)
}
```

**Identifiers & keywords:**
```go
func (s *Scanner) scanIdentifier() {
    for isAlphaNumeric(s.peek()) {
        s.advance()
    }

    text := s.source[s.start:s.current]
    tokenType := LookupIdent(text)
    s.addToken(tokenType)
}
```

## Testing Patterns

**Table-driven test:**
```go
tests := []struct {
    source   string
    expected TokenType
}{
    {"var", VAR},
    {"123", NUMBER},
}

for _, tt := range tests {
    scanner := NewScanner(tt.source)
    // ... test logic
}
```

## Remember

- Always check `isAtEnd()` before peeking
- Track line numbers for multiline constructs
- Use `match()` for conditional consumption
- Keyword lookup happens after identifier scan
```
